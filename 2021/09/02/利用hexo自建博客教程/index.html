hexo +github搭建个人博客

​ 前前后后大概花了两天的时间，目前一个简洁的个人博客搭建完成，后续还会继续完善。现在写个文章做个阶段总结，后续如果有更新的地方，会及时补充。本博客基于Hexo框架，使用hexo-theme-keep主题。

Hexo 初级搭建

1.安装Git

Git是目前世界上最先进的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。Git的作用是将本地的网页文件传到github上。

输入cmd命令台输入git –version，出现版本号说明安装成功。

2.安装node.js

Hexo是基于node.js编写的，所以需要安装一下node.js和里面的npm工具。

windows： 到Node.js官网下载.exe文件，安装选项全部默认。安装好之后，按Win+R打开cmd命令提示符，输入node -v和npm -v，若出现版本号，则说明安装成功。

4.安装Hexo

前面git和nodejs安装好后，就可以安装hexo了。在git bash你可以先创建一个文件夹MyBlog，用来存放自己的博客文件，然后cd到这个文件夹下。这个文件夹就是你要搭建博客的地方。

前面1到4步基本不会出现啥问题，第五步经常会出行莫名其妙的问题 ，执行 hexo init 会要求你执行 npm install ,到这里就很多人安装不上npm ，网上找的其他教程都没啥用，最好的方法执行下面代码即可安装成功.
